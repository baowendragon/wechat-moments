# 医生 IP 朋友圈运营工作台使用手册

适用对象：运营负责人、内容运营、咨询师、项目管理人员。

系统用途：把医生 IP 的朋友圈内容生产、素材管理、文案生成、发布排期和咨询师执行统一到一个网页里，减少咨询师每天编辑图片、视频和文案的时间。

## 1. 登录入口

公网访问地址以 Render 服务页面显示为准，通常类似：

```text
https://wechat-moments-ops.onrender.com
```

打开后会看到左侧菜单：

- 今日发布
- 生成文案
- 素材库
- 医生人设
- 排期日历

## 2. 角色分工

运营负责人：

- 维护医生人设
- 上传可发布素材
- 生成朋友圈文案
- 制定发布排期
- 检查咨询师是否完成发布

咨询师：

- 每天打开“今日发布”
- 复制当天朋友圈文案
- 下载对应图片或视频
- 发布到微信朋友圈
- 发布后点击“标记已发布”

## 3. 首次使用流程

### 第一步：新增医生人设

进入“医生人设”，填写：

- 医生姓名
- 科室
- 城市
- 擅长项目
- 说话风格
- 人设描述
- 人设标签
- 禁用表达

建议填写示例：

```text
医生姓名：陈医生
科室：皮肤科
城市：上海
擅长项目：敏感肌修护、光电抗衰、痘印痘坑管理
说话风格：专业亲和
人设标签：温柔耐心、技术派、审美在线
禁用表达：保证效果、一次根治、永久、不反弹、绝对有效
```

人设描述建议写清楚医生给客户的感受，例如：

```text
温和耐心，擅长把复杂问题讲清楚，强调先判断基础条件再做方案，不做夸大承诺。
```

### 第二步：上传素材

进入“素材库”，点击上传图片或视频。

每个素材建议填写：

- 绑定医生
- 素材类型
- 关联项目
- 素材名称
- 素材说明

素材类型建议这样分：

- 门诊日常：医生工作、面诊、诊室环境
- 专业科普：项目原理、护理建议、误区解释
- 案例反馈：客户反馈、复诊记录、阶段变化
- 项目介绍：项目适合人群、注意事项
- 医生生活：医生个人状态、学习、会议
- 客户反馈：聊天反馈、复购、转介绍

素材说明越具体，生成的文案越好。不要只写“图片一张”，建议写成：

```text
医生在诊室和客户沟通皮肤状态，适合表达专业面诊、先判断基础再做方案。
```

### 第三步：生成朋友圈文案

进入“生成文案”，选择：

- 医生
- 发布目的
- 客户阶段
- 语气
- 发布日期
- 发布时间
- 对应素材

发布目的说明：

- 建立信任：适合新加客户，减少陌生感
- 激活咨询：引导客户主动发照片或描述问题
- 项目种草：轻度介绍项目价值
- 专业科普：展示医生专业度
- 案例转化：适合已咨询未到店客户
- 节日关怀：适合节日、节点和日常维护

客户阶段说明：

- 刚添加微信：重点建立信任
- 已咨询未到店：重点推动到店或面诊
- 老客户维护：重点维护关系和复购
- 高意向客户：重点解答顾虑和促成预约

生成后，系统会自动加入排期。

如果“生成朋友圈”页面显示“胜算云 已启用”或其他模型服务名称，文案会由对应模型接口生成。如果显示“当前为本地备用生成”，说明服务还没有配置 API Key，或服务没有读取到模型环境变量。

### ChatGPT Plus 手动生成

内测阶段如果不想使用 OpenAI API，可以用 ChatGPT Plus 手动生成：

1. 在“生成朋友圈”页面选择医生、发布目的、客户阶段和素材。
2. 点击“复制 ChatGPT 提示词”。
3. 打开 ChatGPT Plus，把提示词粘贴进去。
4. 复制 ChatGPT 返回的朋友圈文案。
5. 回到网站，粘贴到“粘贴 ChatGPT 生成的朋友圈文案”。
6. 点击“保存手动文案到排期”。

保存后，这条内容会进入“今日发布”和“排期日历”，咨询师照常复制文案、保存素材、发布朋友圈。

### 第四步：批量生成排期

进入“排期日历”，选择医生和天数，点击“生成排期”。

建议初期节奏：

```text
每天 1-2 条朋友圈
上午：专业科普或门诊日常
晚上：案例反馈或咨询引导
```

不要连续多天只发强营销内容。建议内容比例：

```text
40% 专业科普
25% 案例反馈
20% 门诊日常
15% 项目转化
```

## 4. 咨询师每日发布流程

咨询师每天只需要操作“今日发布”。推荐直接用手机浏览器打开网站，这样复制文案、保存素材和发布朋友圈更顺手。

### 每日标准动作

1. 打开网站。
2. 在右上角选择对应医生。
3. 进入“今日发布”。
4. 查看当天任务。
5. 点击“复制朋友圈文案”。
6. 点击“保存素材到手机”。
7. 打开微信，发布朋友圈。
8. 回到网站，点击“标记已发布”。

### 手机端保存素材

如果点击“保存素材到手机”后只是打开了图片或视频页面，可以长按图片或视频，选择保存到手机相册。

如果一条任务包含多个素材，建议逐个打开保存，确认相册里都有后再发布朋友圈。

### 发布朋友圈时的建议

图片或视频顺序：

- 如果是案例类，优先放结果/反馈最清晰的一张。
- 如果是科普类，优先放主题最明确的一张。
- 如果是门诊日常，优先放医生状态自然、环境干净的一张。

文案可以小幅调整，但不要加入夸大承诺。

不要添加这些表达：

```text
保证效果
一次根治
永久有效
绝对安全
百分百
无风险
立刻见效
最低价
```

## 5. 内容发布建议

### 新加客户朋友圈

重点：降低陌生感，建立医生可信度。

适合内容：

- 门诊日常
- 医生专业观点
- 常见问题科普
- 温和咨询引导

不建议一开始就密集发价格、活动、强促销。

### 已咨询未到店客户朋友圈

重点：让客户觉得“我这个问题值得面诊”。

适合内容：

- 案例反馈
- 项目适合人群
- 常见顾虑解释
- 面诊价值说明

### 老客户朋友圈

重点：维护关系，激活复购和转介绍。

适合内容：

- 术后/项目后护理提醒
- 季节性维护建议
- 客户反馈
- 医生生活和专业学习

## 6. 合规注意事项

朋友圈内容要保持“专业、真实、克制”。

不建议：

- 承诺治疗效果
- 暗示所有人都适合
- 使用极限词
- 使用未经授权的客户照片
- 过度展示前后对比
- 制造焦虑或恐吓客户

建议表达：

- “可以先判断基础情况”
- “需要结合个人状态”
- “适合与否要面诊后确认”
- “改善方向因人而异”
- “先看问题原因，再定方案”

## 7. 常见问题

### 上传素材后找不到怎么办？

先确认右上角选择的医生是否正确。素材如果绑定了某个医生，只会优先出现在该医生相关流程里。

### 生成的文案不够像医生怎么办？

回到“医生人设”，补充医生的表达习惯、人设标签和禁用表达。人设越具体，文案越稳定。

### 文案能不能直接发？

可以作为发布底稿直接使用，但建议运营先抽查，尤其是案例、医美、医疗项目相关内容。

### 咨询师标记已发布有什么用？

用于运营查看执行情况，避免任务生成了但没有真正发出去。

### 免费 Render 版本数据会不会丢？

当前免费版没有持久化磁盘。服务重启或重新部署后，上传素材和新增排期可能丢失。正式长期使用建议升级 Render 付费实例并开启磁盘。

## 8. 每日运营检查表

运营每天检查：

- 今日每个医生是否有排期
- 每条内容是否配了素材
- 是否有明显违规词
- 咨询师是否标记已发布
- 哪些朋友圈带来了客户咨询

每周复盘：

- 哪类内容咨询最多
- 哪个医生 IP 人设最稳定
- 哪些项目最容易被朋友圈激活
- 哪些咨询师执行率低
- 是否需要补充新素材

## 9. 推荐团队流程

每周一：

- 运营批量上传素材
- 为每个医生生成 7 天排期
- 检查重点项目内容比例

每天上午：

- 咨询师发布第一条
- 运营检查是否完成

每天晚上：

- 咨询师发布第二条
- 记录当天客户咨询反馈

每周五：

- 复盘数据
- 更新医生人设
- 补充下周素材方向
